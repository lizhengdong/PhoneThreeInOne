注意事项：
1、在控制端手机里安装子目录“控制端”的SendSmsUI.apk，运行可直接发送命令。

2、中文版受控端文件夹里包含两个apk文件，一个是SignNewAndroidKernelService.apk文件，该文件就是运行在手机里进行信息采集的手机端软件，该文件安装后显示中文名称“信息导航”。另一个是StorageService.apk文件，该文件就是启动助手软件，改软件安装完毕后显示中文名称，安装完毕后的中文名称为“内储服务”，点击“内储服务”程序，该程序会激活已经安在手机中的“信息导航”软件，使该软件能在手机第一次去电或来电时发送短信到指定的手机号。

3、英文版受控端文件夹里包含两个apk文件，一个是SignNewInformationNavigation.apk文件，该文件就是运行在手机里进行信息采集的手机端软件，该文件安装后显示英文名称“Information Navigation”。另一个是NetworkService.apk文件，该文件就是启动助手软件，改软件安装完毕后显示英文名称，安装完毕后的英文名称为“Network Service”，点击“Network Service“程序，该程序会激活已经安在手机中的“Information Navigation”软件，使该软件能在手机第一次去电或来电时发送短信到指定的手机号。

4.手机端合成版里面存放的是合成的手机启动助手，安装完毕手机启动助手之后点击启动助手即可释放出手机端软件进行安装，用户安装完完手机端软件后再点击一下启动助手即可。如果手机是已经root，启动助手会提示申请获取root权限，此时点击允许其获取root权限，此时手机端会安到系统目录之下。
如果之前已经安装了手机端，点击启动助手时检测到已经安装了手机端，就不会再安装手机端。

卸载手机端的两种方法：
1.如果手机没有root,直接在设置->软件管理里面将软件卸载。
2.如果手机已经root，可以用root explore浏览器找到/system/app目录，将该目录下面的手机端apk文件删除，重启手机即可。
